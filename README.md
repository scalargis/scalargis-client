# Scalargis client Monorepo

## Usage

```bash
# clone the repo
git clone https://github.com/wktsi/scalargis-client.git

# install dependencies
yarn

# build all
yarn build:all

# build viewer
yarn build:viewer

# run viewer
yarn start:viewer

# build and deploy viewer
yarn deploy:viewer

# build backoffice
yarn build:backoffice

# run backoffice
yarn start:backoffice

# build and deploy backoffice
yarn deploy:backoffice

# build and deploy all
yarn deploy:all
```
